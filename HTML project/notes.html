General information

The web is written in html which is a code language

Whenever you want to see a web page the browser sends a request to the server with http (which stands for hypertext transfer protocol).
In this way the server sends the information of the web page in fragments of html language.

Whenever you're disinging a web page the name of the archive should be .html in order to be read in a browser

HTML meaning

HTML stand for hypertext markup language.

Hypertex:is text that can have links or references.

Markup: is the part of the html that gives character to the text so that it won't be plain.

Things to keep in mind

Syntax:Grammar rules of language

Formalism:Computer are literal, so be accurated while writing cause they don't correct mistakes or interpret words.

Nesting:some pieces of code go inside another.

Important fact: you can use a code inside another but you have to close them in the correct order, otherwise it won't work.

Elements

Tags elements are commands that are use in the coding so that you can give some shape or structure to the text.

To use them you have to put the correct command between angle brakets to start, this is called opening tag and then,
to finish the command, angle brackets with a bar in the middle and the command behind the bar, this is called closing tag.

Some examples of commands are:

  • u = underline
  • p = paragraph
  • sub = subscript
  • sup = superscript
  • mark = to highlight
  • em = emphasis (italics)
  • strong = boldface

Void elements or empty elements

Are tags that affect a text without affecting any word and because of this they don't need a closing tag.
For example: br(break) between angle brackets or img (image) between angle brackets.

Types of elements

The elements are classify into two types:

  1. Block elementes
    • p = paragraph
    • ol,ul = lists
    • h1,h2 = headlines
    • video = video player
    • div = generic block
    • table = table element
  2. Inline elements
    • br = line break
    • em, strong = emphasis
    • mark = highligthing
    • a = hypertext link
    • img = image
    • span = generic inline

Bare in mind that for usind these elements you shuold put them betwwen angle brackets.

Lists

When you want to do a list there are 2 types of elements you can use

  1. ol = ordered list(which use numbres)
  2. ul = unorder list (which use bullets)

It's important that whenever you use them you use a closing tag.

There's a very important element that's only used inside the lists "li", these element has to be written in fornt of each item on the list,
but it doesn't need to have a closing tag, although you can put one.
This is because "li" can only be used inside a list and so the browser assume that whenever another li element appear it has to became another point in the list.
The same happens to the "p" element which doesn't need a closing tag, but is recommended that you put it though.
It's important that this does not apply to the ol or ul elements these are elemtents that need to be closed.

Documents

An html document has two parts the head and the body.

  1. Head: Is where we put centain pieces of important information about the document.
    • Title: It tells the browser what to put on the tab where it displays the document.
    • Style: These gives the font and the color to the text.
    • Script: These are the programs that run in the browser.
    • Html lang: This is the language in which is written the page, such us en for english or fr for french.
    • Meta name: This is the metadata for search for search engines.

  2. Body: Is where the contents goes.
    • p = paragraph
    • img = images
    • ol, ul = lists
    • h1 to h6 = headings
    • a = links
